# Чотирьохрозрядний мультиплексор 8-в-1 на Verilog

Цей проект реалізує типовий вузол комп'ютера — мультиплексор з 8 інформаційними входами (4 біти кожен).

## Параметри:
- **Входи:** 8 шин по 4 біти (D0-D7)
- **Адреса:** 3 біти (A2, A1, A0)
- **Вихід:** 1 шина 4 біти (Q)
